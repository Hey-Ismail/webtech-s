# Web Development Portfolio and Lab Tasks

This repository contains multiple web development projects and lab exercises, focusing on responsive design, modern JavaScript syntax, and DOM manipulation.

## 📂 Project Structure

### `portfolio/`

A retro-styled personal student portfolio website built using HTML, CSS, and Vanilla JavaScript.

- 🌙 **Features:** Dark/Light mode toggle implementation.
- ⌨️ **Dynamic UI:** Includes a smooth typing animation effect on the hero section.
- 📱 **Responsive Grid:** Flexbox and CSS Grid applied for optimal project showcasing.
- ✉️ **Contact Form:** Interactive form with basic JavaScript validation.

### `midterm/`

Contains academic lab tasks (`LabTask-01` through `LabTask-09`) covering foundational Web Development concepts.

- Explores HTML structure, CSS layouts (Flexbox/Grid), and JavaScript interactivity.
- Includes supplementary notes and test files.

### 📄 Other Files

- `testTask.html` - A miscellaneous front-end testing task layout.

## 🛠️ Built With

- **HTML5** (Semantic elements)
- **CSS3** (Custom properties, Media queries)
- **JavaScript** (ES6+, DOM Manipulation)

## 🚀 Getting Started

Simply open `portfolio/index.html` or any of the `midterm/LabTask-*.html` files in any modern web browser to view the projects locally. No build tools or package managers are required!
